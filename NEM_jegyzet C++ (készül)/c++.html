Tömbök

A tömbök névvel és sorszámmal ellátott változók.

Egy dimenziós tömb

Ha a tömb elemre hivatkozáskor egy indexet használunk, akkor egy dimenziós (vektorról) beszélünk.

1) int het[7];
2) het[6] = 0;
3) string nap[3] = {"hétfő", "kedd", "szerda"};
4) cout << nap[0] << endl;
5) cout << het[6] << endl;
6) cout << nap[het[6]];
  1. soranban egy egész típusú, 7 elemű tömböt dekralálunk.
  2. sorban az előzőleg dekrarált tömb utolsó elemét megadjuk, 0 lesz az értéke.
  3. sorban láthatunk példát arra, hogyan kell dekrarációkor kezdőértéket adni a tömbelemeknek.
  4. sorban kiíratjuk a nap tömb első elemét, ami 0. indexű: hétfő.
  5. soran kiíratjuk a het tömb utolsó elemét, ami azért nem a 7-es mert a 7 elemű tömbindexelése 0-tól 6-ig tart. A kiírt elem: 0.
  6. sorban arra látunk példát, hogy az egyik tömb (nap) indexe lehet egy másik tömb (het) eleme. A kiírt elem: hétfő.

Két dimenziós tömb

Ha az adataink táblázatos elrendezésüek, azaz olyanok, mint egy többoszlopos és többsoros fiókos szekrény, tárolásukra kétdimenziós tömböt (mátrixot) alkalmazunk.

int kep[100][50];
kep[0][49] = 62;
int tabla[3][3] = { { 0, 1, 2 }, { 2, 1, 1 }, { 0, 2, 1 } };
char figura[3] = { '.', 'o', 'x'};
cout << kep[0][49] << endl;
cout << tabla[2][1] << endl;
cout << figura[tabla[2] [1]];
  1. sorban dekrarálunk egy 100 soros és 50 oszlops egész típusú tömböt.
  2. sorban az előzőleg dekralált tömb 1. sorámak (0.index), utolsó oszlopában (49.index) található elemnek a 62 értéket adjuk
  3. - 5. sorban értékadás.

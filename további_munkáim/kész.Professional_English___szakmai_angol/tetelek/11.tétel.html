I am working for a Software developing company. Our company has received a request (kérés) to supply the conference with all IT devices and possibilities.

You have to register, study the program of the conference, get to know the speakers and performers (előadók) and give online feedback and evaluation (értékelés). We work out evaluation forms. We need such a solution (megoldás) which ensures it on mobile phones. I would like to show you the advantages and disadvantages of mobile applications and websites.

From registration to reporting (jelentésküldés), networking to notifications (értesítés), EventMobi does it all. We built a platform with everything in one easy-to-use app.

Registration & Check-In App

We have created a registration site built right into your event app to register anywhere, anytime, on any device. We can keep track (nyomon követ) of everyone’s arrival using the check-in app. It is a seamless (zökkenőmentes) experience for attendees (belépők); a great way for you to save time and money.

Mobile Event App

It is an event app that makes it simple to find your event information quickly, online or offline. Available in all 4 major app stores, as well as compatible with all mobile and desktop browsers.

Audience Response System (hallgatói válasz rendszer)

It gives everyone a voice with live polls (élő szavazások), surveys (közvéleménykutatás) and session evaluation (értékelés) built into your event app. You can engage attendees and watch response rates skyrocket (kimagasló válaszadási arány).

Live Display (élő közvetítés)

It is a digital signage (felirat) that goes beyond social walls, integrating with your event app, alerts, gamification, and sponsor messages to give your attendees the information they need.

Mobile Event Games

It can energize, educate, and engage participants (résztvevők). It inspires action and enhance (fokoz) /increase the on-site experience (élmény).

Reporting (jelentés)

You can go beyond download rates and discover what attendees really want with the Engagement Dashboard (kapcsolatfal). Comprehensive (átfogó) data sets simplified into powerful reports helping you demonstrate your report. With live analytics (elemzések) or on-demand (igény szerinti) report downloads we’ve got your reporting needs covered.

Summary (összegzés)

Mobile apps have many advantages. Firstly, everybody has got a smart phone. The mobile phones have internet access, we can join at WiFi network or we can use our internet connection on our mobile. The advantage of mobile phone is that you are available anytime and anywhere. Everybody has got an own web browser on his/her mobile. So we have created a responsive (alkalmazkodó) web design for mobile screen. The web application is better, because you can use software without installation.

A mobile application software is useful because, all your events are integrated seamlessly in one application. After registration you can find the events. This software also has gamification. The next part of the software is polling and surveys.

Another advantage of mobile application is that through the app you can watch the conference as a live display. The next part of application is event analytics where you can write your comments, poll, feedback, complete questionnaires and make surveys.

This software is available at any type of computers, mobile phones, tablets, desktops and laptops. The software adapts all kinds of stores, for example App Store, Google Play (Play Store), BlackBerry World and Windows Store.
